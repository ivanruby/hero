What is it

Hero.js is a Javascript library that provides offline storage and geolocation functionalities using HTML5 APIs (Web Storage and Geolocation).

It exposes 2 main objects, hero.db and hero.geo
* hero.cache and hero.nav are the other objects in the library but might be removed from the library in the future.
